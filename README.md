## ūüĒī Render Deployment Guide

> ‚ö†ÔłŹ **IMPORTANT ‚ÄĒ Read before deploying on Render**

**Step 1 ‚ÄĒ Fork this repo**
- Go to the top of this page and click **Fork**
- Do NOT deploy directly from the original repo

**Step 2 ‚ÄĒ Customize your `main.py`**
- You **MUST** modify `main.py` before deploying (see AI Customization below)
- Direct copies from the same repo will trigger Render's duplicate detection

**Step 3 ‚ÄĒ Connect to Render**
1. Go to [dashboard.render.com](https://dashboard.render.com)
2. Click **New +** ‚Üí **Web Service**
3. Select **"Build and deploy from a Git repository"**
4. Connect your GitHub and select your **forked** repo
5. Render auto-detects `render.yaml` ‚ÄĒ all settings are pre-configured

**Step 4 ‚ÄĒ Set Environment Variables**

Add these in Render's **Environment** tab:

| Variable | Description | Required |
|---|---|---|
| `BOT_TOKEN` | Bot token from [@BotFather](https://t.me/BotFather) | ‚úÖ |
| `ASSISTANT_SESSION` | Pyrogram string session | ‚úÖ |
| `API_ID` | From [my.telegram.org](https://my.telegram.org) | ‚úÖ |
| `API_HASH` | From [my.telegram.org](https://my.telegram.org) | ‚úÖ |
| `OWNER_ID` | Your Telegram user ID | ‚úÖ |
| `SEARCH_API_URL` | Leave blank for default | ‚ĚĆ |

**Step 5 ‚ÄĒ Deploy**
- Click **"Create Web Service"** and wait for the build to finish

/active

2. Open [youtube.com](https://youtube.com) and log in
3. Click the extension ‚Üí Export cookies for `youtube.com`
